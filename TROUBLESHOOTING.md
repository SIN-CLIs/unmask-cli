## 2026-05-02: Fix-Statistiken

**Änderungen pro Datei:**

## Permission denied

→ Systemeinstellungen → Datenschutz → Bedienungshilfen → Terminal aktivieren

## Build fails

→ `swift build` / `npm run build` / `pip install -e .`  
→ Prüfe .gitignore und fehlende Dependencies

## Tool not found

→ `which <tool>` prüfen  
→ `make install` ausführen
